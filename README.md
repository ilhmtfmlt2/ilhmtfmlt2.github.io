# 濮阳石油化工职业技术学院成绩查询系统

---
## 简介

**濮阳石油化工职业技术学院成绩查询系统**

用于帮助学生轻松快捷地查询他们的学术成绩

该系统利用学校官方网站提供的API接口，直接获取学生成绩无需繁琐步骤

该系统只用于成绩查询，不涉及其他功能或数据

用户的隐私和数据安全是我们的首要关注，系统不会存储或滥用用户信息
